# Gestion des Utilisateurs - API Node.js

## Description
Ce projet est une interface web simple et moderne permettant de gérer des utilisateurs via une API Node.js.

L'application permet :
-  Ajouter un utilisateur
- Afficher la liste des utilisateurs
- Mettre à jour automatiquement la liste après ajout

---

## Technologies utilisées

- HTML5
- CSS3 (Design moderne avec Flexbox)
- JavaScript (Vanilla JS)
- API REST (Node.js)
